## 🌟 项目简介

**Smart Food Chef** 是一款旨在解决现代人“吃什么”与“怎么吃”双重痛点的智能应用。通过整合食材库、营养学算法与优质视频资源，它能将你冰箱里的平凡食材，转化为科学、健康的饮食方案。

### 核心痛点解决：
- **食材积压**：不知道现有的食材能做什么菜？勾选一下，AI 帮你秒匹配。
- **营养焦虑**：不知道这顿饭热量高不高？实时看板，精准计算四大宏量营养素。
- **健身目标**：想减脂/增肌不知道吃多少？个性化 TDEE 自动计算并提供摄入建议。

---

## ✨ 核心功能

### 1. 智能食材匹配
- 分类勾选荤、素、主食、调料。
- 基于本地数据驱动的菜谱推荐系统。
- 一键跳转至 **Bilibili** 高质量视频教学，手把手教你做菜。

### 2. 实时营养看板 (Real-time Analytics)
- 实时动态计算选中食材的总热量（Calories）、蛋白质、脂肪与碳水。
- 采用 **Chart.js** 实现可视化饼图，配比一目了然，辅助科学决策。

### 3. 个性化 TDEE 计算器
- 集成 **Mifflin-St Jeor** 科学算法，计算每日总能量消耗（TDEE）。
- 提供“减脂/维持/增肌”三种模式的针对性建议，让饮食更有计划性。
