# 🥑 Smart Food Chef | 智能厨师

> **基于 AI 驱动的个性化健康饮食管理系统**
>

---

## 🌟 项目简介

**Smart Food Chef** 是一款旨在解决现代人“吃什么”与“怎么吃”双重痛点的智能应用。通过整合食材库、营养学算法与优质视频资源，它能将你冰箱里的平凡食材，转化为科学、健康的饮食方案。

### 核心痛点解决：
- **食材积压**：不知道现有的食材能做什么菜？勾选一下，AI 告诉你。
- **营养焦虑**：不知道这顿饭热量高不高？实时看板，精准计算。
- **健身目标**：想减脂/增肌不知道吃多少？个性化 TDEE 自动计算。

---

## ✨ 核心功能

### 1. 智能食材匹配
- 分类勾选荤、素、主食、调料。
- 秒级匹配相关菜谱。
- 一键跳转至 **Bilibili** 高质量视频教学。

### 2. 实时营养看板 (Real-time Analytics)
- 实时计算选中食材的总热量（Calories）、蛋白质、脂肪与碳水。
- 采用 **Chart.js** 实现数据可视化饼图，配比一目了然。

### 3. 个性化 TDEE 计算器
- 基于 **Mifflin-St Jeor** 科学算法。
- 根据身高、体重、年龄、性别及活动强度，计算每日总能量消耗。
- 提供“减脂/维持/增肌”三种模式的针对性建议。

## 🛠️ 技术栈

本项目采用了创新的 **AI 协同开发 (AI-Assisted Development)** 模式：

- **前端框架**: [Next.js 14](https://nextjs.org/) (App Router)
- **样式处理**: [Tailwind CSS](https://tailwindcss.com/)
- ** UI 组件**: [v0.dev](https://v0.dev/) 玻璃拟态风格组件
- **数据可视化**: [Chart.js](https://www.chartjs.org/)
- **逻辑开发**: [Trae](https://www.trae.ai/) (AI-native IDE)
- **视觉设计**: [Leonardo.ai](https://leonardo.ai/) (Logo 设计)
